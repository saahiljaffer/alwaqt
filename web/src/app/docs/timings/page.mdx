# Timings

---

## List prayer timings for a specific date {{ tag: 'GET', label: '/api/timings/:date' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a list of all timings for a specific date (default: today).

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/api/timings/:date">

    ```bash {{ title: 'cURL' }}
    curl -G https://alwaqt.app/api/timings/:date
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "sunrise": "5:36",
      "imsak": "3:31",
      "sunset": "8:58",
      "dhuhr": "1:17",
      "fajr": "3:36",
      "maghrib": "9:13"
    }
    ```

  </Col>
</Row>
